# Level-Up Gamer - Manual de Usuario y Proyecto

## Descripción General
Level-Up Gamer es una plataforma de e-commerce para productos gaming, desarrollada como proyecto semestral full-stack. Incluye frontend en React + Vite y backend en Spring Boot, con autenticación JWT y base de datos MySQL/PostgreSQL.

## Estructura del Proyecto
- `admin-react/`: Aplicación web para usuarios y administración.
- `backend-spring/`: API REST y lógica de negocio.

## Funcionalidades Principales
- Registro y login de usuarios con roles (admin, vendedor, cliente).
- Gestión de productos, stock y pedidos.
- Sistema de puntos y descuentos automáticos para clientes DUOC.
- Panel de administración para usuarios y productos.
